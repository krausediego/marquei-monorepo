# Regras para o Assistente

- Responda SEMPRE em português brasileiro
- Quando fizer alterações em código, explique claramente:
  - O QUE foi alterado
  - POR QUÊ foi alterado
  - COMO a solução funciona
- Use comentários no código quando apropriado
- Se houver múltiplas soluções, explique os trade-offs
- Seja conciso mas completo nas explicações
- Priorize clareza sobre brevidade
